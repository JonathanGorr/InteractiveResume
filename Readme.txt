This package contains WebGL template suitable for Facebook games.

Features:
- logo and progress indicator
- downloading progress in Mb
- full-frame

Usage:
- make sure you have "Assets/WebGLTemplates/FacebookTemplate/" folder in your project
- open player settings (File > Build Settings... > Player Settings...)
- switch to WebGL
- open "Resolution and Presentation" tab
- select WebGL template
- build you project
- host on server / run locally in Firefox
